# Kassenstand

Personal finance flow tracker. Works entirely in the browser — no installation needed.

## 🚀 How to use

### Option 1: GitHub Pages (recommended)
1. Fork this repository
2. Go to Settings → Pages → Source: Deploy from branch → `main` / `/ (root)`
3. Visit `https://yourusername.github.io/kassenstand`

### Option 2: Open locally
Just open `index.html` in any browser — no server needed.

## 💾 Data storage
Your data is saved in your browser's localStorage. It stays on your device.
Use **Settings → Export JSON** to make backups, and **Import JSON** to restore.
